# TechDashboard

- .NET 8 WPF 暗色科技风 Dashboard 示例
- 左侧可折叠导航栏（图标 + 文本）、主题切换（Dark / Light）
- MVVM（MainViewModel） + ObservableObject 基类

## 运行

1. 需要 .NET 8 SDK
2. 打开终端到项目根目录
   - `dotnet build`
   - `dotnet run --project TechDashboard/TechDashboard.csproj`
或在 Visual Studio 中打开 `TechDashboard.sln` 并运行。
