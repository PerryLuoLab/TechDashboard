# Screenshots

This directory contains screenshots of the TechDashboard application.

## Required Screenshots

Please add the following screenshots to showcase the application:

1. **main-interface.png** - Main dashboard interface showing the overall layout
2. **theme-dark.png** - Application with Dark theme applied
3. **theme-light.png** - Application with Light theme applied
4. **theme-bluetech.png** - Application with Blue Tech theme applied
5. **language-support.png** - Language selection interface showing all 5 languages
6. **navigation-panel.png** - Navigation panel features (collapsed/expanded states)

## Screenshot Guidelines

- **Format**: PNG format recommended for best quality
- **Size**: Recommended resolution: 1920x1080 or 1280x720
- **Naming**: Use lowercase with hyphens (e.g., `main-interface.png`)
- **Content**: Ensure screenshots clearly show the features being demonstrated

## How to Take Screenshots

1. Run the application: `dotnet run`
2. Navigate to different pages and themes
3. Use Windows Snipping Tool or any screenshot tool
4. Save screenshots in this directory with the names listed above
5. Update the README.md files to reference these images
